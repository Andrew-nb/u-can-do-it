## ✨ 功能

- 😴 **睡眠打卡** — 记录每天的入睡时间，用图表查看自己的作息趋势
- ✅ **习惯打卡** — 支持每日打卡和每周打卡，灵活设定目标
- 📅 **缺卡记录** — 日历视图一目了然，用颜色标记每天的完成情况

### 📱 添加到手机主屏幕

把它放到桌面上，用起来就像一个 App：

- **安卓手机**：浏览器打开上面的链接 → 点右上角菜单 → 选「添加到主屏幕」
- **苹果手机**：用 Safari 打开上面的链接 → 点底部的分享按钮 → 选「添加到主屏幕」

## 💡 小贴士

- 本应用以**凌晨 4:00** 作为一天的分界线，所以熬夜到凌晨 3 点打卡，算的还是前一天
- 每周从**周一**开始计算
- 数据保存在你的浏览器本地，换设备或清除浏览器数据会丢失，记得珍惜你的打卡记录哦 😊
